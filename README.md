## What is Examar?

Examar is an Augmented Reality based student assessment application that 
places students in simulated real-world scenarios to evaluate their 
knowledge — rather than testing them through traditional pen-and-paper 
examinations.

Instead of answering abstract questions on a page, students step into a 
virtual environment through their iPhone camera and apply their knowledge 
practically. A teacher observes and scores them in real time alongside 
the system's automated evaluation.

## Features

- 🏪 **AR Supermarket Portal** — A fully immersive AR environment built 
  with ARKit that places students inside a virtual supermarket
- 🧮 **Scenario Based Assessment** — Students solve real-world math 
  problems within the AR environment
- ✅ **Instant Validation** — The system validates answers at each stage 
  and prompts students to retry if incorrect
- 📊 **Result Screen** — Detailed performance summary on completion
- 📝 **Teacher Observation Notes** — Educators can record qualitative 
  observations alongside the automated score
- 📱 **iPhone Native** — Built with SwiftUI and ARKit for a smooth, 
  modern iOS experience

## How to Run

1. Clone the repository
```bash
git clone https://github.com/RohanKatyayani/Examar.git
```

2. Open `Examar.xcodeproj` in Xcode 26

3. Connect your iPhone and select it as the build target

4. Press **CMD + R** to build and run

5. Trust the developer certificate on your iPhone:
   **Settings → General → VPN & Device Management → Trust**

## How it Works

Student opens Examar
↓
Enters name and grade
↓
AR Portal opens via iPhone camera
↓
Student navigates virtual supermarket
↓
Answers math questions at each section
↓
System validates answers in real time
↓
Result screen generated
↓
Teacher adds observation notes
↓
Final combined assessment saved
